# Sandbox Improvements
A set of tools to make sandbox more powerful and easy to use.

### Features
- Hotkeys for various hooks that may happen during a game, allowing cards that rely on hooks to be used in sandbox.
- Map selection menu in sandbox, similar to Unbound's map toggle menu. Works as an alternative to the "/" command.
- Card spawning menu in sandbox, similar to Unbound's card toggle menu. Works as an alternative to the chat.
